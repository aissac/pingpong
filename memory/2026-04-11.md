### Lossless-Claw Upgrade: 0.4.0 → 0.8.0
- **Bug fixed:** "No API provider registered for api: ollama" error that was causing LCM summarization to fail and fall back to truncation

- **pi-coding-agent:** Made by Mario Zechner (libGDX author). It's the core runtime engine for OpenClaw — handles LLM provider registration, session management, tool execution, extension system. OpenClaw = pi-coding-agent + gateway server + channel integrations + memory systems.
- **Status:** Zero errors since upgrade at 13:35 EDT on April 10. Compaction hasn't triggered yet (context at ~35% of 203K token limit, needs 75%). Quality of future summaries should improve since summarization actually works now instead of truncating.
