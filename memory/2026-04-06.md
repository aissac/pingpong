### The BTC vs ETH Difference

| Asset | LEG1 Entry | LEG2 Exit | Combined | Result |

| ETH | YES @ 53¢ | NO @ 42¢ | 95¢ | ✅ +5¢ |
| BTC | YES @ 48¢ | NO @ 90¢ | 138¢ | ❌ -38¢ |

**Root Cause:** Inverse correlation broke for BTC
- ETH NO dropped after YES crash → cheap hedge
- BTC NO rallied after YES crash → expensive hedge

**The Fix:** Add stop-loss cap at 120¢

```rust
const STOP_LOSS_MAX_COMBINED: f64 = 120.0;

// Forced hedge at 5min, but ONLY if combined ≤ 120¢
let stop_loss_ok = forced_hedge && combined <= STOP_LOSS_MAX_COMBINED;

// Skip if combined > 120¢ - better to lose LEG1 only
if forced_hedge && combined > STOP_LOSS_MAX_COMBINED {
    eprintln!("⚠️ STOP-LOSS SKIPPED combined={:.0}¢ > 120¢ - L: {}¢", combined, price);
}
```
